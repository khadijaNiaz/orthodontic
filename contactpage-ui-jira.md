# JIRA TASK -- Contact Page UI Issues

## Title:
UI Issues on Contact Page -- Form Layout, Alignment & Map Issues

## Issue Type:
Bug

## Priority:
High

## Description:
Several UI issues affecting usability on the Contact Page.

## UI Issues Identified:
1. Google Map iframe misaligned.
2. Form fields uneven in height and spacing.
3. Submit button too small on mobile.
4. Missing input validation messages.
5. Header overlapping content when scrolling.
6. Placeholder text not aligned properly.

## Expected Behaviour:
- Proper alignment, spacing, and typography.
- No text overlapping.
- Consistent layout and components.
- Fully responsive on 1280px and above.

## Steps to Reproduce:
1. Open the page on a 1280px or 1366px laptop.
2. Scroll through sections.
3. Observe spacing, alignment, and layout issues.
