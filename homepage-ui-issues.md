# JIRA TASK -- Home Page UI Issues

## Title:

UI Issues on Home Page -- Layout, Spacing, Typography & Alignment Fixes

## Issue Type:

Bug

## Priority:

High

## Description:

Several UI inconsistencies and layout issues are observed on the Home
Page. These issues affect readability, visual hierarchy, brand
consistency, and user experience across desktop screens.

## UI Issues Identified:

1.  Heading typography overlap and misalignment.
2.  Icons misaligned in header section.
3.  Excessive white space in hero banner area.
4.  Content overlapping in multiple sections.
5.  Misaligned CTA buttons.
6.  Uneven card heights and icon alignment.
7.  Incorrect spacing in "Meet the Orthodontist" section.
8.  Patient Journey columns not aligned.
9.  Testimonial spacing and star alignment broken.
10. Footer spacing and alignment issues.

## Expected Behaviour:

-   Proper alignment, spacing, and typography.
-   No text overlapping.
-   Consistent layout and components.
-   Fully responsive on 1280px and above.

## Steps to Reproduce:

1.  Open the homepage on a 1280px or 1366px laptop.
2.  Scroll through sections.
3.  Observe spacing, alignment, and layout issues.
